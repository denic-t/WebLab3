## 2. Сравнение интерфейсов (PrevWebLab vs WebLab3)

Целью было максимально приблизить интерфейс новой лабораторной (JSF) к предыдущей (JSP/HTML).

### Визуальное сходство
*   **CSS**: Файл стилей `style.css` был полностью перенесен из `PrevWebLab`, что обеспечило идентичную цветовую схему, шрифты и отступы.
*   **Шапка**: Идентичное содержание и стиль.

### Элементы управления
| Элемент | PrevWebLab (JSP) | WebLab3 (JSF) | Комментарий |
| :--- | :--- | :--- | :--- |
| **Ввод X** | Checkboxes | Radio Buttons (`h:selectOneRadio`) | В JSF радио-кнопки логичнее для выбора *одного* значения. Визуально стиль сохранен. |
| **Ввод Y** | Text Input | Text Input (`h:inputText`) | Полное соответствие. |
| **Ввод R** | Text Input | Spinner (`p:spinner`) | Улучшение UX. Позволяет вводить число или выбирать стрелками. |
| **График** | SVG + JS | SVG + JS | Логика отрисовки и взаимодействия полностью перенесена в `main.js`. |
| **Таблица** | HTML `<table>` | JSF `<h:dataTable>` | Стили таблицы сохранены, функционал идентичен. |
| **Пагинация**| Custom JS | JSF Backend Logic | В `WebLab3` реализована серверная пагинация (First, Prev, Next, Last) с выбором размера страницы, как в оригинале. |
| **Модальное окно** | Custom JS | Custom JS + JSF | Реализовано всплывающее окно с результатом проверки, аналогичное `PrevWebLab`. |

### Вывод
Интерфейс `WebLab3` визуально и функционально практически неотличим от `PrevWebLab`, при этом внутренняя архитектура полностью переведена на JSF, соблюдая все требования варианта (за исключением СУБД, которая была мигрирована на PostgreSQL).
